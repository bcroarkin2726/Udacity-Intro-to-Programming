HTML is reasonably straightforward

HTML stands for HyperText Markup Language

HTML consists of tags and elements that are broken up in the following format:
<tag>Contents<tag>
The tags tell the browsers the type of the element to be used.

W3Schools is a great resource for when you need additional help

For instance, I got these great definitions below for inline and block-level elements there!

Block-level Elements

A block-level element always starts on a new line and takes up the full width available
(stretches out to the left and right as far as it can).

Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary.
